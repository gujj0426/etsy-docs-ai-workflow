// 安装说明
// ────────────────────────────────────────────────────────────────

// 第一步：加载插件到 Chrome/Edge
// 1. 打开 Chrome，地址栏输入：chrome://extensions/
// 2. 右上角开启「开发者模式」
// 3. 点击「加载已解压的扩展程序」
// 4. 选择本目录（browser_extension）

// 第二步：访问腾讯文档
// 打开 workFlowTest 智能表格（file_id: tbfaTE）
// https://docs.qq.com/dop-api/get/sheet?docId=KFoUkmaZFqLP

// 第三步：上传图片测试
// 在「客户原图」字段上传任意图片
// 插件会自动：
// 1. 拦截上传请求，提取图片
// 2. 上传到腾讯云 COS（etsy-ai-images-1405462135）
// 3. 弹出提示显示 CDN URL

// 第四步：配置验证（可选）
// 点击浏览器右上角插件图标 → 弹窗中修改/确认 COS 配置
// 可用「手动上传测试」功能单独测试

// ────────────────────────────────────────────────────────────────
// 文件说明：
// manifest.json  - 插件配置清单（Manifest V3）
// background.js  - 后台脚本，负责 COS 上传（TC3 V3 签名）
// content.js     - 注入到 docs.qq.com，拦截上传、写入字段
// popup.html     - 插件弹窗界面
// popup.js       - 弹窗逻辑（配置保存 + 手动测试）
// icons/         - 图标文件（16x16 / 48x48 / 128x128）
// README.md      - 本文件
// ────────────────────────────────────────────────────────────────
// 已知限制：
// 1. 需 HTTPS 环境或 localhost，file:// 协议下 service_worker 有兼容问题
// 2. 腾讯文档页面结构更新可能导致 DOM 查找失效，届时更新 content.js
// 3. 手动上传测试可绕过自动拦截，直接验证 COS 连接是否正常
